What’s new in v2.4 (focus: “3D data orb” depth + admin transparency spine)

VISUAL / MOTION
- OrbitControls-based slow camera circling (parallax depth). In admin mode you can drag/zoom.
- Faster ring rotation (default ringSpeedMult=1.80) with slider in Admin Tools.
- Physical glass-like core (MeshPhysicalMaterial + env reflections) for real specular depth.
- Subtle generated environment map (CanvasTexture) + minimal key/rim/ambient lights (no shadows).

ADMIN UI / OPERATIONS
- Admin overlay updated (higher contrast, more professional labels).
- “Admin tools” panel: ring speed, camera orbit speed, fog density, particles, outer shell, ref arcs.
- Visual config persistence via localStorage (export/import/reset).
- Always-on audit log (localStorage-backed) + export JSON/CSV.
- Admin snapshot polling + 3D graph layer: snapshot → generic { nodes, links }.

ITERATION 4.3 SUPPORT (Admin transparency + data spine)
