How it works
Two incentives: Skill (rank) + Activity (paid spend). Weekly adds longer competition cycles.
Skill Paid-only leaderboards (per game, per metric) drive the Skill pool payout.
Activity Global paid AC spend drives Activity/PRO/Lottery pools with diminishing returns.
Multi-metric Each game can define multiple metrics (score/time/etc). One metric = one board.
Identity PRO nickname/avatar may appear while membership is active and the NFT is SBT-locked.

Fairness + Diversity
Different games can rank on different metrics without breaking payout logic.
  • Skill boards can be multi-metric (score, time, accuracy, streak).
  • Eligibility is paid-only for payouts; free/promo runs stay visible for hype.
  • Game-aware scoring: time metrics are lower-is-better, score metrics are higher-is-better.
  • Anti-farm: Activity rewards are based on paid AC spent (diminishing returns).
  • Identity: PRO nickname/avatar can show on leaderboards while active & SBT-locked.
